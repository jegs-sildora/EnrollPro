# REQUIREMENTS

## In Scope (v1)
1. **Online Admission Portal:** Public-facing multi-step wizard for applications (`/apply`).
2. **F2F Admission:** Registrar-facing equivalent for walk-in applicants.
3. **Registrar Dashboard:** Stat cards, Recharts visualizations, and recent activity feed.
4. **Enrollment Management:** Application inbox with dynamic two-path approval workflow (Regular vs SCP).
5. **SIMS (Student Information Management System):** Permanent searchable records and academic history tracking.
6. **Teacher Management:** Teacher profiles and section assignments.
7. **Grade Level & Sectioning:** Configurable grade levels, strands, and max capacities.
8. **Academic Year Configuration:** Dynamic dates for early reg and regular enrollment; dynamic SCP setups.
9. **Audit Trail:** Immutable activity logging for significant actions.
10. **Email System:** Automated, asynchronous Nodemailer + Resend SMTP notification system.

## Out of Scope (v2/Future)
- Grading Systems
- Class Scheduling / Timetables
- DepEd Official Form Generation (SF1, SF4) automated printing
- Cashiering / School Fees / Billing
- SMS Notifications
- Student / Parent Portal Login (Applicants interact via public form only)
- Multi-School / Multi-Branch Instances (Single-school installation only)
