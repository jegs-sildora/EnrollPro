================================================================================
DATA FLOW DICTIONARY & TABLE MAPPING (SENIOR ARCHITECT AUDIT)
================================================================================

PROCESS: P4.0 Submit Early Registration
-> Inbound from Learner: [BEEF Annex 1 Profile] (Noun: Demographics, LRN, contact info packet)
<- Inbound from DB: [Active Window Check] (Reads D9 to verify Phase 1 is open)
<- Inbound from DB: [Valid Grade Levels] (Reads D20 for JHS hierarchy validation)
<- Outbound to Learner: [Phase 1 Tracking Number] (Noun: 10-digit monitoring identifier)
-> Outbound to DB:
   - D1: learners (Persists permanent identity)
   - D2: early_registration_applications (Persists Phase 1 metadata)
   - D3: application_addresses (Persists geographic data)
   - D4: application_family_members (Persists parent/guardian profiles)
   - D10: audit_logs (Security trail)

PROCESS: P5.0 Submit Official Enrollment
-> Inbound from Learner: [BEEF Annex 2 Modalities] (Noun: Learning modality and track specialization packet)
<- Inbound from DB: [Active Window Check] (Reads D9 to verify Phase 2 is open)
<- Inbound from DB: [Pre-populated Profile] (Reads D1, D3, D4 to eliminate redundant entry)
<- Outbound to Learner: [Enrollment Status] (Noun: Notification of official placement state)
-> Outbound to DB:
   - D5: enrollment_applications (Persists official BOSY intent)
   - D6: enrollment_previous_schools (Persists SF10 history)
   - D7: enrollment_program_details (Persists SCP track specifics)
   - D10: audit_logs (Security trail)

PROCESS: P5.1 Submit Continuing Learner Confirmation
-> Inbound from Learner: [Return Intent] (Noun: Consent packet for returning students)
<- Inbound from DB: [Existing Application] (Reads D5 to link confirmation to previous profile)
<- Outbound to Learner: [Confirmation Slip] (Noun: Printable verification of return intent)
-> Outbound to DB:
   - D5: enrollment_applications (Updates intent with intakeMethod='CONFIRMATION_SLIP')
   - D10: audit_logs (Security trail)
